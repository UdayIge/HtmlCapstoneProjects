HTML Capstone Projects

Friendship Zone

A Simple Website to Make Friends

The goal of this project is to create a simple and neat website where people can register and keep a list of their friends. It helps in learning how to build web pages using just HTML.

AutoMob-MechanicApp

A Car Service Web Application

The aim of this project is to create a basic website for an automobile service center where users can view services, book appointments, and get service-related information. It helps beginners learn how to structure multiple HTML pages for a real-world use case.
